# React Hooks State Management

## Description
A small React application demonstrating state management using React Hooks (`useState` and `useEffect`). This app allows users to add, view, and delete items from a list.

## What's Built
- A simple UI to manage a list of items.
- State management using React Hooks.
- Local storage for persistence.

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/react-hooks-state.git
   cd react-hooks-state
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm run dev
   ```
4. Open your browser and go to `http://localhost:3000`.

## API Docs
- **GET /api/items**: Retrieve all items.
- **POST /api/items**: Add a new item.
- **DELETE /api/items/:id**: Delete an item by ID.

## Environment Variables
No environment variables are required.
